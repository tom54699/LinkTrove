# Change: Update Search UI to Subtle Pill Design

## Why
為了提升搜尋體驗，將原本的兩段式搜尋（下拉選單 -> 彈窗）簡化為直接開啟全螢幕搜尋模態視窗。同時優化視覺風格，使其更符合現代化的設計美學與組織主題色連動。

## What Changes
- **UI 觸發器**：使用圓角膠囊狀的 **Subtle Pill** 按鈕，Hover 時背景會變色（`var(--panel)`）。
- **快捷鍵支援**：
  - `Ctrl+F` / `Cmd+F`：攔截瀏覽器預設搜尋，開啟全域搜尋彈窗。
  - `Ctrl+K` / `Cmd+K` 與 `/`：同樣支援開啟彈窗。
- **彈窗設計**：
  - 加寬至 `max-w-3xl`，提供更寬敞的閱讀空間。
  - 字體大小與間距微調，提升易讀性（結果標題 15px，描述 11-12px）。
- **搜尋邏輯優化**：
  - **排除無群組項目**：搜尋結果會自動過濾掉沒有 `subcategoryId` 的卡片，確保顯示結構完整。
- **最近搜尋 (Recent Search)**：
  - 在輸入框為空時，顯示最近 **10 筆** 搜尋紀錄。
  - 紀錄包含 **相對時間戳記**（如：5m ago）。
- **自動清空**：關閉彈窗（ESC 或點擊背景）時，自動清空搜尋輸入框內容。
- **組織顏色連動**：
  - 彈窗內的結果計數文字、輸入框光標（Caret）顏色。
  - **卡片跳轉高亮**：徹底修復顏色衝突，點擊結果後，目標卡片的高亮邊框與光暈完全使用組織代表色。
- **鍵鼠同步操作**：
  - 滑鼠懸停（onMouseMove）與鍵盤選中狀態完全同步，樣式統一使用 `var(--card)` 背景，消除分離感。
  - 項目右側顯示動作提示（↵ 開啟 / ↵ 搜尋）。

## Impact
- **Affected Specs:** `search`
- **Affected Code:** `src/app/ui/SearchBox.tsx`
