## MODIFIED Requirements
### Requirement: 全域搜尋介面
系統必須（SHALL）提供全域搜尋介面，透過點擊 Subtle Pill 樣式按鈕或快捷鍵啟動模態視窗。

#### Scenario: 使用 Ctrl+F 啟動搜尋
- **WHEN** 使用者按下 Ctrl+F（Windows/Linux）或 Cmd+F（Mac）
- **THEN** 系統攔截瀏覽器預設行為，顯示全螢幕搜尋模態視窗（寬度 max-w-3xl）
- **THEN** 搜尋輸入框自動聚焦

#### Scenario: 搜尋過濾與完整性
- **WHEN** 系統執行搜尋
- **THEN** 必須過濾掉沒有 `subcategoryId` 的卡片
- **THEN** 搜尋結果僅顯示歸屬於特定群組的書籤

#### Scenario: 顯示最近搜尋記錄
- **GIVEN** 搜尋模態視窗開啟且輸入框為空
- **THEN** 顯示最近 10 筆搜尋記錄列表，包含相對時間戳記（如 5m ago）
- **WHEN** 使用者點擊一筆紀錄
- **THEN** 輸入框自動填入該字句並執行搜尋

#### Scenario: 搜尋結果操作與導覽
- **GIVEN** 搜尋結果列表已顯示
- **WHEN** 使用者使用方向鍵或滑鼠移動到某個項目
- **THEN** 該項目背景變更為 `var(--card)`，並顯示右側操作提示（↵ 開啟）
- **THEN** 滑鼠與鍵盤焦點必須保持完全同步
- **WHEN** 使用者按下 Enter 鍵或點擊滑鼠
- **THEN** 系統導航至該書籤位置並關閉彈窗
- **THEN** 目標卡片顯示使用組織代表色的高亮框與光暈效果（必須移除預設綠色）

#### Scenario: 關閉彈窗並清空
- **GIVEN** 搜尋模態視窗已開啟，且輸入框有內容
- **WHEN** 使用者按下 ESC 鍵、點擊背景或執行導航
- **THEN** 搜尋模態視窗關閉
- **THEN** **下次開啟時搜尋輸入框必須為空**

#### Scenario: 組織主題色連動
- **THEN** 模態視窗內部的結果計數、輸入框光標應顯示當前組織的代表色
