## ADDED Requirements

### Requirement: 匯入命名對話框防誤關閉
系統必須（SHALL）確保 HTML/Toby 匯入流程中的命名對話框，在使用者編輯名稱欄位時不會因拖曳路徑跨到遮罩而誤關閉。

#### Scenario: HTML 匯入命名欄位拖曳不誤關閉
- **GIVEN** 使用者開啟「匯入 HTML（新集合）」對話框
- **AND** 對話框中存在可編輯名稱欄位
- **WHEN** 使用者在名稱欄位內按下滑鼠並拖曳
- **AND** 於背景遮罩放開滑鼠
- **THEN** 對話框保持開啟
- **AND** 欄位內容保持不變

#### Scenario: Toby 匯入命名欄位拖曳不誤關閉
- **GIVEN** 使用者開啟「匯入 Toby」對話框
- **AND** 對話框中存在可編輯名稱欄位
- **WHEN** 使用者在名稱欄位內按下滑鼠並拖曳
- **AND** 於背景遮罩放開滑鼠
- **THEN** 對話框保持開啟
- **AND** 欄位內容保持不變

#### Scenario: 純遮罩點擊仍可關閉匯入命名對話框
- **GIVEN** 使用者開啟任一匯入命名對話框
- **WHEN** 使用者在背景遮罩完成一次純點擊（`mousedown` 與 `mouseup` 皆在遮罩）
- **THEN** 對話框關閉
- **AND** 不觸發匯入執行
