# Capability: Global Search

## ADDED Requirements

### Requirement: 搜尋結果分頁與自動載入
系統必須（SHALL）在搜尋結果數量超過單頁上限時，先顯示第一頁結果並於使用者捲動接近底部時自動載入下一頁，直到全部結果載入完成。

#### Scenario: 接近底部載入更多
- **GIVEN** 搜尋結果共有 55 筆
- **WHEN** 使用者輸入關鍵字並看到前 20 筆結果
- **THEN** 使用者捲動接近結果列表底部時（例如距底部 200px 內），自動載入下一頁結果
- **THEN** 重複操作直到顯示全部 55 筆

### Requirement: 簡繁互通搜尋
系統必須（SHALL）支援繁體與簡體互通搜尋，無論資料或查詢字串為繁/簡，都能正確匹配結果。

#### Scenario: 繁體資料與簡體查詢互通
- **GIVEN** 卡片標題為「前端開發」
- **WHEN** 使用者搜尋「前端开发」
- **THEN** 系統仍顯示該卡片

#### Scenario: 簡體資料與繁體查詢互通
- **GIVEN** 卡片標題為「网站优化」
- **WHEN** 使用者搜尋「網站優化」
- **THEN** 系統仍顯示該卡片

### Requirement: 分頁結果提示
系統必須（SHALL）在搜尋結果超過單頁上限時，顯示「已載入 / 總數」的提示資訊，讓使用者了解目前載入進度。

#### Scenario: 顯示已載入與總數
- **GIVEN** 搜尋結果共有 42 筆
- **WHEN** 使用者初次看到前 20 筆結果
- **THEN** 搜尋結果區域顯示「20 / 42」或等價提示（包含總數）
